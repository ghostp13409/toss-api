---
Toss-api is a blazing fast, Vim-inspired TUI and CLI API client built in Rust. It brings the power of full-featured API testing into your terminal, allowing you to seamlessly manage collections, environments, and requests without leaving your keyboard.
---
Smart Parsing automatically extracts API endpoints directly from your source code. (Supported Frameworks: Spring, Rails, Laravel, ASP.Net, Express, FastAPI, Django, etc.)
Navigate effortlessly using familiar Vim keybindings. 
Seamlessly transition between a rich TUI dashboard and quick CLI commands.
---
Directly import Postman collections to maintain your existing workflows. 
Features a robust REST client with full support for Params, Headers, Auth, and various Body types, complete with syntax highlighting for responses and environment variables.
---
Ideal for backend developers and terminal power users who want a distraction-free, keyboard-driven API testing workflow. Perfect for quickly exploring new APIs or testing local services without the overhead of heavy electron apps.
